### <a name="step-4-interpret-the-result"></a>ステップ 4: 結果を解釈する

私たちが送信したい元の__メッセージ__は、次のとおりです。

$$ \ケット{\psi} = \{0}アルファ\ケット +{1} \ベータ\ケット $$

受け取った状態が意図された状態になるように、この状態に__そこに__量子ビットを取得する必要があります。 

* 測定して00の結果を得た場合、3番目の量子ビットは、__状態__$(\α\ket{0} +\β\ket){1}です。 これは意図された__メッセージ__であるため、変更は必要ありません。
* 測定して01の結果を得た場合、3番目の量子ビットは、__状態__$(\α\ket{1} +\β\ket){0}です。 これは意図した__メッセージ__とは異なりますが、NOT ゲートを適用すると、目的の状態 $(\α\ケット{0}+\{1}ベータ\ケット)$が得られます。
* 測定して10の結果を得た場合、3番目の量子ビットは、__状態__$(\α\ケット{0}-\ベータ\ケット{1})です。 これは意図した__メッセージ__とは異なりますが、Z ゲートを適用すると、目的の状態 $(\α\ケット{0}+\{1}ベータ\ケット) $ が表示されます。
* 測定して11の結果を得た場合、3番目の量子ビットは、__状態__$(\α\ケット{1}-\ベータ\ケット{0})です。 これは意図した__メッセージ__とは異なりますが、NOT ゲートの後に Z ゲートを適用すると、目的の状態 $(\α\ケット{0}{1}+\ベータ\ケット)$が得られます。

要約すると、測定する場合、最初の量子ビットが1であれば、Zゲートが適用されます。 測定する場合、2 番目のクビットが 1 の場合は、NOT ゲートが適用されます。

### <a name="step-0-definition"></a>ステップ 0: 定義
テレポーテーションを行う際には、送信したい__メッセージ__と、そのメッセージをどこに送信したいのかを知る必要__があります__。 このため、引数として 2 つの量子ビットを与えられた新しいテレポート操作を定義し、`msg`と`there`:

```qsharp
operation Teleport(msg : Qubit, there : Qubit) : Unit {
```

また、`here``using`ブロックで達成する qubit を割り当てる必要があります。

```qsharp
    using (here = Qubit()) {
```

### <a name="step-1-create-an-entangled-state"></a>手順 1: 絡み合った状態を作成する
その後、と 操作を使用して、`here`絡`there`み合った@"microsoft.quantum.intrinsic.h"ペア@"microsoft.quantum.intrinsic.cnot"を作成できます。

```qsharp
        H(here);
        CNOT(here, there);
```

### <a name="step-2-send-the-message"></a>手順 2: メッセージを送信する
次の $\演算子名 {CNOT}$ と $H$ ゲートを使用して、メッセージの量子ビットを移動します。

```qsharp
        CNOT(msg, here);
        H(msg);
```

### <a name="step-3--4-measuring-and-interpreting-the-result"></a>ステップ 3 & 4: 結果の測定と解釈
最後@"microsoft.quantum.intrinsic.m"に、測定を実行し、必要なゲート操作を実行して、`if`必要な状態を取得します。

```qsharp
        // Measure out the entanglement
        if (M(msg) == One)  { Z(there); }
        if (M(here) == One) { X(there); }
```

### <a name="step-5-restarting-the-qubit-register"></a>手順 5: qubit レジスタを再起動する

Q# の各操作の最後に、qubits を状態 $\ket{0}$にする必要があります。 すべての量子ビット@"microsoft.quantum.intrinsic.reset"をゼロ状態に再起動するために使用することができ、これは私たちの操作を終了します。

```qsharp
        Reset(msg);
        Reset(here);
        Reset(there);
    }
}
```
