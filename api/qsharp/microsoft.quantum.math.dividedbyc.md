2つの入力の商を返します。

```qsharp
function DividedByC (a : Microsoft.Quantum.Math.Complex, b : Microsoft.Quantum.Math.Complex) : Microsoft.Quantum.Math.Complex
```


## <a name="input"></a>入力

1番目の入力 $a $ に分割されます。


### <a name="b--complex"></a>b: [Complex](xref:Microsoft.Quantum.Math.Complex)
