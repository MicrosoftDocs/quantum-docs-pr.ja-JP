数値が別の数値より大きい場合にのみ true を返します。

```qsharp
function GreaterThanI (a : Int, b : Int) : Bool
```

比較する最初の値。


### <a name="b--int"></a>b: [Int](xref:microsoft.quantum.lang-ref.int)

比較する2番目の値。



## <a name="output--bool"></a>出力: [Bool](xref:microsoft.quantum.lang-ref.bool)
