# <a name="microsoftquantumoracles-namespace"></a>Oracles 名前空間

この名前空間は、フェーズの推定や振幅増幅などのクォンタムアルゴリズムでの用途によって、さまざまな種類の oracles を装飾するためのユーザー定義型を提供します。
