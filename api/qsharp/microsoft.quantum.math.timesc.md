2つの入力の積を返します。

```qsharp
function TimesC (a : Microsoft.Quantum.Math.Complex, b : Microsoft.Quantum.Math.Complex) : Microsoft.Quantum.Math.Complex
```


## <a name="input"></a>入力

### <a name="a--complex"></a>a: [Complex](xref:Microsoft.Quantum.Math.Complex)
